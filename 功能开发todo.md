# 功能开发 TODO 清单

## ✅ 已完成功能 (Completed Features)

### 1. 系统架构与基础
- [x] **Tauri 跨平台框架搭建**：基于 Rust + Web 技术的桌面应用架构。
- [x] **前端 UI 框架**：集成 HTML + Tailwind CSS，支持响应式布局。
- [x] **深色/浅色模式**：支持界面主题切换（UI 适配完成）。
- [x] **前后端通信**：实现 Tauri IPC (`invoke`) 机制，打通 Rust 与 JS 数据交互。

### 2. 系统健康监控
- [x] **CPU 监控**：
    - 后端：使用 `sysinfo` 实时采集 CPU 全局使用率。
    - 前端：动态数值显示与自动刷新。
- [x] **内存监控**：
    - 后端：实时采集已用/总内存数据。
    - 前端：可视化进度条与数值显示。

### 3. 自定义服务监控 (NAS/Lab)
- [x] **服务管理 (CRUD)**：
    - 支持添加、编辑、删除自定义服务监控项。
    - **数据持久化**：配置自动保存至 `~/.monitor-dashboard/services.json`。
- [x] **健康检查引擎**：
    - **Web 服务**：支持 HTTP/HTTPS 状态码检测。
    - **TCP 服务**：支持端口连通性检测。
    - **指标采集**：实时计算响应时间 (Latency) 和在线状态。

### 4. 界面交互
- [x] **页面导航**：实现“概览”与“服务监控”页面的无刷新切换。
- [x] **交互反馈**：服务状态指示灯、加载状态、错误提示。
